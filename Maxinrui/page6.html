Life's Director
AI生成视频小程序

项目时间

2024.09 - 2024.12

项目角色

产品经理 / 前端开发

项目成果

30+种子用户验证

项目介绍

一款面向中老年用户的语音输入与AI生成平台，将口述故事自动转化为结构化时间轴与短视频，帮助用户留存家庭记忆资产。

定义"语音输入 → AI结构化 → 视频生成"核心闭环，完成竞品分析与产品需求文档（PRD），参与demo设计与生成。

技术架构

前端技术栈

使用React + TypeScript构建前端界面，确保代码类型安全和可维护性，提供流畅的用户交互体验。

后端技术栈

Node.js + Express搭建后端服务，配合Supabase（pgvector）存储用户数据与向量嵌入，支持RAG记忆检索。

AI能力集成

集成阿里云DashScope（Qwen + embedding）实现RAG记忆检索与补全，豆包Seedance模型实现文本到视频的自动生成。

项目亮点

  • 语音输入转时间轴
  • AI自动生成短视频
  • 老年用户体验优化
  • RAG记忆检索
  • 30+种子用户验证

使用工具

React TypeScript Node.js Supabase 阿里云DashScope
